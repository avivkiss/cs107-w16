Submodules¶

crypto.simulator.base_sim module¶

class crypto.simulator.base_sim.BaseSim(game, adversary)[source]¶

This is the base simulator class that all simulators inherit from. It has the base constructor which all simulators use.

Parameters:
  • game – This is the specific instantiation of the game you wish to perform a simulation of.
  • adversary – This is the adversary you would like to run against

the game.

crypto.simulator.bind_sim module¶

class crypto.simulator.bind_sim.BINDSim(game, adversary)[source]¶

Bases: crypto.simulator.base_sim.BaseSim

This simulator was written to be used with GameBIND. It simulates the game with an Adversary and allows you to compute an approximate advantage.

Parameters:
  • game – This is the specific instantiation of the game you wish to perform a simulation of.
  • adversary – This is the adversary you would like to run against

the game.

compute_advantage()[source]¶

Adv = Pr[Bind => true]

Returns:Approximate advantage computed using the above equation.
compute_success_ratio()[source]¶

Runs the game 10 times and computes the ratio of successful runs over total runs. This is fewer runs than usual because RSA math is expensive/slow.

Returns:successes / total_runs
run()[source]¶

Runs the game with the adversary provided to the constructor.

Returns:True for success and False for failure.
